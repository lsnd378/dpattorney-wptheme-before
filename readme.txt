== Description ==

D Pongkor & Partners adalah tema WordPress premium yang dirancang khusus untuk firma hukum Dion Pongkor & Partners. Tema ini menampilkan desain modern dan profesional dengan tema gelap yang elegan, animasi yang halus, dan dukungan penuh untuk SiteOrigin Page Builder.

Fitur Utama:
- Desain modern dengan tema gelap yang elegan
- Animasi scroll yang halus
- Dukungan SiteOrigin Page Builder
- Custom Post Types untuk Team, Practice Areas, Insights, dan Careers
- Custom Taxonomies untuk pengorganisasian konten
- Theme Customizer untuk pengaturan mudah
- Responsive design untuk semua perangkat
- SEO friendly
- Translation ready

== Installation ==

1. Upload tema ke direktori `/wp-content/themes/`
2. Aktifkan tema melalui menu 'Appearance' di WordPress
3. Konfigurasi pengaturan tema melalui Customizer
4. Install dan aktifkan plugin SiteOrigin Page Builder (opsional tapi direkomendasikan)

== Custom Post Types ==

Tema ini menyediakan 4 Custom Post Types:

1. **Team Members** - Untuk menampilkan anggota tim firma hukum
2. **Practice Areas** - Untuk menampilkan area praktik hukum
3. **Insights** - Untuk artikel dan publikasi
4. **Job Openings** - Untuk lowongan pekerjaan

== Custom Taxonomies ==

1. **Team Types** - Partner, Senior Associate, Associate
2. **Practice Categories** - Kategori untuk area praktik
3. **Insight Categories** - Kategori untuk artikel
4. **Job Categories** - Kategori untuk lowongan pekerjaan

== Theme Customizer ==

Akses pengaturan tema melalui Appearance > Customize:

- Hero Section - Atur konten hero homepage
- Stats Section - Atur statistik yang ditampilkan
- Contact Information - Atur informasi kontak
- Social Media - Atur link media sosial
- Footer - Atur konten footer

== Widget Areas ==

1. **Footer Widget Area 1** - Area widget di footer
2. **Footer Widget Area 2** - Area widget di footer
3. **Blog Sidebar** - Sidebar untuk halaman blog

== Menu Locations ==

1. **Primary Menu** - Menu utama navigasi
2. **Footer Menu** - Menu di footer
3. **Legal Menu** - Menu untuk link legal (Privacy Policy, Terms, dll)

== Recommended Plugins ==

- SiteOrigin Page Builder - Untuk membuat halaman dengan drag-and-drop
- SiteOrigin Widgets Bundle - Widget tambahan untuk Page Builder
- Yoast SEO - Untuk optimasi SEO
- WPForms - Untuk membuat form kontak
